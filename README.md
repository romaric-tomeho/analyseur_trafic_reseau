# Analyseur de Trafic Réseau (Simulé)

## Description
Outil de simulation de trafic réseau permettant de modéliser des nœuds réseau,
simuler l'envoi de paquets de données et identifier les goulots d'étranglement.

## Prérequis
- Python 3.10+
- pip

## Installation

### 1. Cloner le dépôt
```bash
git clone https://github.com/romaric-tomeho/analyseur_trafic_reseau.git
cd analyseur_trafic_reseau
```

### 2. Créer et activer l'environnement virtuel
```bash
python -m venv venv

# Windows
venv\Scripts\activate

# Linux/Mac
source venv/bin/activate
```

### 3. Installer les dépendances
```bash
pip install -r requirements.txt
```

## Lancement
```bash
python main.py
```
